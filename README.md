# amplify-react-tutorial

このリポジトリは、Amplifyを使ったReactアプリのハンズオンです。

作成するアプリは、認証機能付きのToDoアプリです。

## 構築に必要なもの

* node.js
* yarn

## 概要

AWS Amplifyを使うと様々な機能が爆速で開発できます。

本チュートリアルで、Amplifyを使った開発プロセスのさわりを体感してみましょう。
